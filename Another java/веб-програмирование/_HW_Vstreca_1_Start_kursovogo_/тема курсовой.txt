------ web-приложение учебного заведения ------ 


 
---------------------- ТЗ ----------------------

- Разработать функционал для 3х видов пользователей  
  (учитель, ученик и администратор).

- Создать для приложения user-friendly UI/UX дизайн

- Реализовать функцию сохранения/загрузки данных в/из	
  базы данных

- Реализовать заполнения отчетности, выставления оценок и т.д
  для преоподователей и администраторов.

- Реализовать на front-end части функции сортировки данных по
  разным критериям в зависимости от типа данных  
  (например сортировка дз по дате или оценке и 
   сортировка учеников по алфовитному порядку или среднему балу)

- По возможности внедрить не навязчивых анимаций в дизайн 
  (где это будет уместно)



----------------- Технологиии -----------------

- front-end часть будет разработываться на 
  js, html, css, React Library + Angular Framework


- back-end часть будет разработываться на 
  ASP.NET или чистом php
