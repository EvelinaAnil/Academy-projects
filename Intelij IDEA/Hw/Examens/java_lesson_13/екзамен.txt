Spring MVC + Hibernate
Скріншоти робочої програми в корень проекту.

Створити застосунок для створення викладачів для академії Шаг.
У викладача є ім'я, прізвище, вік, номер телефону, пошта, технології.
Відобразити список усіх викладачів. Зберігати в б.д. дату та
рік народження, а відображати вік на сьогоднішній день.
Підготувати функціонал для створення нового викладача.
Також підготувати функціонал для додавання технологій
викладачу, котрі він може викладати. Технології можуть 
додаватися або видалятися.
