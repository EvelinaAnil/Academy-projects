Uri - класс 
 
Обеспечивает объектное представление универсального кода ресурсов (URI), а также простой доступ к его частям.

Пространство имен:   System
Сборка:  System (в System.dll)

Иерархия наследования
System.Object
System.Uri

Синтаксис
[SerializableAttribute]
[TypeConverterAttribute(typeof(UriTypeConverter))]
public class Uri : ISerializable

Конструкторы
 Uri(SerializationInfo, StreamingContext)	Инициализирует новый экземпляр класса Uri на основе указанных экземпляров классов SerializationInfo и StreamingContext.
 Uri(String)	Инициализирует новый экземпляр класса Uri с заданным универсальным кодом ресурса.
 Uri(String, Boolean)	Устарело.Инициализирует новый экземпляр класса Uri с указанным универсальным кодом ресурса и явным управлением преобразованием в escape-последовательность.
 Uri(String, UriKind)	Инициализирует новый экземпляр класса Uri с заданным универсальным кодом ресурса.Этот конструктор позволяет указать, содержится ли в строке универсального кода ресурса относительный универсальный код ресурса, абсолютный универсальный код ресурса или неопределенный вариант.
 Uri(Uri, String)	Инициализирует новый экземпляр класса Uri на основе заданного базового универсального кода ресурса и строки относительного универсального кода ресурса.
 Uri(Uri, String, Boolean)	Устарело.Инициализирует новый экземпляр класса Uri на основе заданных базового и относительного универсального кода ресурса с явным управлением преобразованием в escape-последовательность.
 Uri(Uri, Uri)	Инициализирует новый экземпляр класса Uri на основе сочетания заданного базового экземпляра Uri и относительного экземпляра Uri.

Свойства
 AbsolutePath	Возвращает абсолютный путь универсального кода ресурса.
 AbsoluteUri	Возвращает абсолютный универсальный код ресурса.
 Authority	Возвращает для сервера службу доменных имен (DNS) или IP-адрес и номер порта.
 DnsSafeHost	Возвращает не преобразованное в escape-последовательность имя узла, которое безопасно использовать для разрешения DNS.
 Fragment	Возвращает escape-последовательность фрагмента универсального кода ресурса.
 Host	Возвращает компонент узла этого экземпляра.
 HostNameType	Возвращает тип имени узла, заданного в универсальном коде ресурса.
 IdnHost	Международное доменное имя узла, соответствующее RFC 3490, в котором используется Punycode.
 IsAbsoluteUri	Возвращает значение, позволяющее определить, является ли экземпляр Uri абсолютным.
 IsDefaultPort	Возвращает значение, позволяющее определить, является ли номер порта, указанный в универсальном коде ресурса, портом данной схемы по умолчанию.
 IsFile	Возвращает значение, позволяющее определить, представляет ли заданный параметр Uri универсальный код ресурса файла.
 IsLoopback	Возвращает значение, позволяющее определить, ссылается ли заданный Uri на локальный узел.
 IsUnc	Возвращает значение, позволяющее определить, является ли заданный Uri UNC-путем.
 LocalPath	Возвращает представление имени файла в локальной операционной системе.
 OriginalString	Возвращает исходную строку универсального кода ресурса, переданную в конструктор Uri.
 PathAndQuery	Возвращает свойства AbsolutePath и Query, разделенные вопросительным знаком (?).
 Port	Возвращает номер порта данного универсального кода ресурса.
 Query	Возвращает любые сведения запроса, включенные в заданный универсальный код ресурса.
 Scheme	Возвращает имя схемы для данного универсального кода ресурса.
 Segments	Возвращает массив, содержащий сегменты пути, которые образуют указанный универсальный код ресурса.
 UserEscaped	Указывает на то, что строка универсального кода ресурса не была полностью преобразована в escape-последовательность перед созданием экземпляра класса Uri.
 UserInfo	Возвращает имя пользователя, пароль или другие связанные с пользователем сведения, которые соответствуют заданному универсальному коду ресурса.

 CheckSchemeName(String)	Определяет, является ли допустимым указанное имя схемы.
 CheckSecurity()	Устарело. Этот API служит для поддержки инфраструктуры продукт и не предназначен для непосредственного использования из кода. Вызов этого метода ни на что не влияет.
 Compare(Uri, Uri, UriComponents, UriFormat, StringComparison)	Сравнивает указанные части двух универсальных кодов ресурса, используя заданные правила сравнения.
 Equals(Object)	Сравнивает два экземпляра Uri на предмет их равенства.(Переопределяет Object.Equals(Object).)
 Escape()	Устарело. Этот API служит для поддержки инфраструктуры продукт и не предназначен для непосредственного использования из кода. Преобразует все небезопасные или зарезервированные символы в компонент пути, используя шестнадцатеричное представление.
 EscapeDataString(String)	Преобразует строку в ее escape-представление.
 EscapeString(String)	Устарело.Преобразует строку в ее escape-представление.
 EscapeUriString(String)	Преобразует строку универсального кода ресурса в ее escape-представление.
 Finalize()	Позволяет объекту попытаться освободить ресурсы и выполнить другие операции по очистке перед тем, как объект будет утилизирован сборщиком мусора. (Наследуется от Object.)
 FromHex(Char)	Возвращает десятичное значение шестнадцатеричной цифры.
 GetComponents(UriComponents, UriFormat)	Возвращает заданные компоненты текущего экземпляра, используя указанное для специальных знаков escape-преобразование.
 GetHashCode()	Возвращает хэш-код для универсального кода ресурса.(Переопределяет Object.GetHashCode().)
 GetLeftPart(UriPartial)	Возвращает заданную часть экземпляра Uri.
 GetObjectData(SerializationInfo, StreamingContext)	Возвращает данные, необходимые для сериализации текущего экземпляра.
 GetType()	Возвращает объект класса Type для текущего экземпляра. (Наследуется от Object.)
 HexEscape(Char)	Преобразует заданный символ в эквивалентное ему шестнадцатеричное число.
 HexUnescape(String, Int32)	Преобразует шестнадцатеричное представление символа в сам символ.
 IsBadFileSystemCharacter(Char)	Устарело. Этот API служит для поддержки инфраструктуры продукт и не предназначен для непосредственного использования из кода. Возвращает значение, позволяющее определить, содержит ли имя файловой системы недопустимый символ.
 IsBaseOf(Uri)	Определяет, является ли текущий экземпляр Uri основой указанного экземпляра Uri.
 IsExcludedCharacter(Char)	Устарело. Этот API служит для поддержки инфраструктуры продукт и не предназначен для непосредственного использования из кода. Возвращает значение, позволяющее определить, нужно ли преобразовывать указанный символ в escape-последовательность.
 IsHexDigit(Char)	Определяет, является ли указанный символ допустимой шестнадцатеричной цифрой.
 IsHexEncoding(String, Int32)	Определяет, является ли кодировка символа шестнадцатеричной.
 IsReservedCharacter(Char)	Устарело. Этот API служит для поддержки инфраструктуры продукт и не предназначен для непосредственного использования из кода. Возвращает значение, позволяющее определить, является ли заданный символ зарезервированным знаком.
 IsWellFormedOriginalString()	Указывает, является ли формат строки, использовавшейся для создания данного Uri, правильным и не требующим дальнейшего преобразования в escape-последовательность.
 IsWellFormedUriString(String, UriKind)	Указывает, является ли правильным формат данной строки, пытаясь создать на ее основе универсальный код ресурса и проверяя, не требуется ли для нее дополнительное преобразование в escape-последовательность.
 MakeRelative(Uri)	Устарело.Определяет разницу между двумя экземплярами класса Uri.
 MakeRelativeUri(Uri)	Определяет разницу между двумя экземплярами класса Uri.
 MemberwiseClone()	Создает "поверхностную" копию текущего объекта. (Наследуется от Object.)
 Parse()	Устарело. Этот API служит для поддержки инфраструктуры продукт и не предназначен для непосредственного использования из кода. Анализирует универсальный код ресурса текущего экземпляра и проверяет, действительно ли этот код содержит все обязательные части допустимого универсального кода ресурса.
 ToString()	Возвращает каноническое строковое представление заданного экземпляра Uri.(Переопределяет Object.ToString().)
 TryCreate(String, UriKind, Uri)	Создает новый Uri, используя заданный экземпляр String и UriKind.
 TryCreate(Uri, String, Uri)	Создает новый Uri, используя заданные экземпляры базового и относительного String.
 TryCreate(Uri, Uri, Uri)	Создает новый Uri, используя заданные экземпляры базового и относительного Uri.
 Unescape(String)	Устарело. Этот API служит для поддержки инфраструктуры продукт и не предназначен для непосредственного использования из кода. Преобразует указанную строку, заменив все escape-последовательности их представлениями, к которым не было применено escape-преобразование.
 UnescapeDataString(String)	Отменяет преобразование строки в escape-представление.

Поля
 SchemeDelimiter	Определяет символы, отделяющие схему протокола передачи данных от адресной части универсального кода ресурса.Это поле доступно только для чтения.
 UriSchemeFile	Указывает на то, что универсальный код ресурса является указателем на файл.Это поле доступно только для чтения.
 UriSchemeFtp	Указывает на то, что данный универсальный код ресурса доступен по протоколу FTP.Это поле доступно только для чтения.
 UriSchemeGopher	Указывает на то, что данный универсальный код ресурса доступен по протоколу Gopher.Это поле доступно только для чтения.
 UriSchemeHttp	Указывает на то, что данный универсальный код ресурса доступен по протоколу HTTP.Это поле доступно только для чтения.
 UriSchemeHttps	Указывает на то, что данный универсальный код ресурса доступен по протоколу HTTPS.Это поле доступно только для чтения.
 UriSchemeMailto	Указывает на то, что данный универсальный код ресурса представляет адрес электронной почты и доступен по протоколу SMTP.Это поле доступно только для чтения.
 UriSchemeNetPipe	Указывает, что доступ к универсальному коду ресурса осуществляется через схему NetPipe, используемую Windows Communication Foundation (WCF).Это поле доступно только для чтения.
 UriSchemeNetTcp	Указывает, что доступ к универсальному коду ресурса осуществляется через схему NetTcp, используемую Windows Communication Foundation (WCF).Это поле доступно только для чтения.
 UriSchemeNews	Указывает на то, что данный универсальный код ресурса представляет группу новостей Интернета и доступен по протоколу NNTP.Это поле доступно только для чтения.
 UriSchemeNntp	Указывает на то, что данный универсальный код ресурса представляет группу новостей Интернета и доступен по протоколу NNTP.Это поле доступно только для чтения.

Операторы

 Inequality(Uri, Uri)	Определяет, не равны ли значения двух экземпляров Uri.
