IPHostEntry - класс
 
Предоставляет класс контейнеров для сведений об адресе веб-узла.
Пространство имен:   System.Net
Сборка:  System (в System.dll)

Иерархия наследования
System.Object
System.Net.IPHostEntry

Синтаксис
public class IPHostEntry

 IPHostEntry()	Инициализирует новый экземпляр класса IPHostEntry.

Свойства
 AddressList	Возвращает или задает список IP-адресов, связанных с узлом.
 Aliases	Возвращает или задает список псевдонимов, связанных с узлом.
 HostName	Возвращает или задает DNS-имя узла.

Заметки
Класс IPHostEntry связывает имя хоста службы DNS с массивом псевдонимов и массивом соответствующих им IP-адресов.
Класс IPHostEntry используется с классом Dns в качестве вспомогательного класса.

Примеры
В следующем примере демонстрируется запрос базы данных DNS относительно сведений об узле www.contoso.com и возврат сведений в экземпляре IPHostEntry.

IPHostEntry hostInfo = Dns.GetHostByName("www.contoso.com");
