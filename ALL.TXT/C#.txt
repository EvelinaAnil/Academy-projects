
--Особенность классов их можно вызывать повторно!!
===============
private — данные доступны только методам внутри класса и вложенным в него классам;

protected — данные доступны только методам внутри класса (и вложенным в него классам) или в любом из его дочерних классов;

internal — данные доступны только в методах теку-щей сборки;

protected internal — данные доступны только методам вложенного или производного типа класса и любым методам текущей сборки;

public — данные доступны всем методам во всех сборках .
===============
static — используется для объявления статического поля, которое принадлежит классу, а не конкретному объекту, то есть является общим для всех экземпля-ров класса 

const — используется для объявления постоянного поля, то есть значение данного поля не может быть изменено . Поле с модификатором const должно быть обязательно проинициализировано при объявлении поля класса . Константные поля являются неявно ста-тическими,  поэтому  обращение  к  ним  необходимо  осуществлять только через имя типа .
readonly — означает, что поле будет использоваться только для чтения, присвоение значений таким полям разрешается только в конструкторе либо сразу при объявлении .
