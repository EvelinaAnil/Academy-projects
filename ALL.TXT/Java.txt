Чтобы создавать программы на Java, на 
вашем компьютере должны
быть установлены библиотеки классов и 
компилятор javac, а чтобы их
запускать, нужно установить среду Java
Runtime Environment (JRE), которая
поддерживает интерпретатор java. 
Все вышеперечисленные компоненты 
содержатся в пакете Java Platform, 
Standard Edition Development
Kit (JDK), находящемся в свободном доступе.
